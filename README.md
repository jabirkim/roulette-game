# 오프라인 룰렛 앱

정적 HTML/CSS/JS로 만든 유럽식 룰렛 게임입니다. 빌드 도구나 외부 의존성 없이 동작합니다.

## 실행 방법

1. 이 폴더에서 `index.html` 파일을 브라우저로 엽니다.
2. 베팅 금액을 입력합니다.
3. 레드/블랙, 홀짝, 1-18 / 19-36 또는 단일 숫자 베팅을 추가합니다.
4. `스핀` 버튼으로 룰렛을 돌립니다.

## 포함 기능

- 유럽식 룰렛 숫자 범위 `0-36`
- 레드/블랙, 홀짝, 로우/하이, 단일 숫자 베팅
- 잔액, 현재 총 베팅, 베팅 내역, 최근 결과 표시
- 단일 숫자 `35:1`, 이븐 머니 `1:1` 지급 로직
- 모바일에서도 누르기 쉬운 반응형 레이아웃
